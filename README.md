# Car Dashboard
Final project for Embedded Systems for the Internet of Things  [UniTn - 145996]

## Idea
Car Dashboard with speed indicator, street sign and speed limit display.
Future idea -> parking sensors

### Libraries 
- LVGL
- TinyGPS++
- ArduinoJSON
- ...
